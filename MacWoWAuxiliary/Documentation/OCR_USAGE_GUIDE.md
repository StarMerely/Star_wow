# OCR 功能使用指南

## 🎯 两种使用模式

OCR 功能提供两种使用模式，满足不同场景需求。

---

## 模式 1: 🔍 查找并点击（单次）

### 适用场景
- 一次性点击某个按钮
- 完成某个特定操作
- 测试 OCR 识别效果

### 使用步骤

1. **输入查找文字**
   ```
   输入框: 确定
   ```

2. **点击按钮**
   ```
   点击: 🔍 查找并点击
   ```

3. **自动执行**
   - ✅ 自动截屏
   - ✅ 识别文字
   - ✅ 查找目标
   - ✅ 点击一次
   - ✅ 自动停止

### 执行流程

```
点击按钮 → 截屏 (1秒) → 识别文字 → 查找并点击 → 完成
```

### 结果提示

- ✅ 成功：`✅ 成功找到并点击 '确定'`
- ❌ 失败：`❌ 未找到 '确定'`

---

## 模式 2: 🔄 循环查找（持续循环）

### 适用场景
- 持续监控某个按钮并重复点击
- 游戏中自动领取奖励
- 自动化重复操作

### 使用步骤

1. **设置循环间隔**
   ```
   循环间隔: 3 秒
   ```

2. **输入查找文字**
   ```
   输入框: 领取奖励
   ```

3. **开始循环**
   ```
   点击: 🔄 循环查找
   ```

4. **手动停止**
   ```
   点击: ⏸ 停止循环
   ```

### 执行流程

```
开始循环 → 截屏 → 识别 → 未找到？继续循环
                          ↓
                      找到了？点击 → 继续循环...
                      
点击 ⏸ 停止循环 → 停止 ✅
```

### 按钮状态

| 状态 | 按钮文字 | 颜色 |
|------|---------|------|
| 未运行 | 🔄 循环查找 | 蓝色 |
| 运行中 | ⏸ 停止循环 | 红色 |

---

## 💡 功能说明

### 扫描间隔设置

**位置**: 循环间隔 (秒)

**作用**: 
- 仅用于"循环查找"模式
- 控制每次查找的间隔时间
- "查找并点击"模式不使用此设置

**推荐值**:
- **快速响应**: 1-2 秒
- **正常使用**: 3-5 秒
- **省资源**: 5-10 秒

### 点击延迟设置

**位置**: 点击延迟 (秒)

**作用**:
- 用于多个文字依次点击
- 每次点击之间的等待时间

**推荐值**:
- **快速点击**: 0.3-0.5 秒
- **正常点击**: 0.5-1 秒
- **需要加载**: 1-2 秒

## 📝 多文字点击

### 语法

使用分号 `;` 分隔多个文字：

```
确定;开始;继续;完成
```

### 执行顺序

```
1. 查找并点击 "确定"
2. 等待 0.5 秒（点击延迟）
3. 查找并点击 "开始"
4. 等待 0.5 秒
5. 查找并点击 "继续"
6. 等待 0.5 秒
7. 查找并点击 "完成"
```

### 结果显示

```
完成! 成功点击 3/4 个文字
```

表示 4 个文字中成功点击了 3 个。

## ⚙️ 高级技巧

### 1. 提高识别准确度

- ✅ 使用完整的文字（如 "确定" 而不是 "确"）
- ✅ 确保文字清晰可见
- ✅ 避免文字被遮挡
- ✅ 调整屏幕分辨率

### 2. 优化性能

- ✅ 合理设置循环间隔（不要太短）
- ✅ 不需要时及时点击停止按钮
- ✅ 使用"查找并点击"而不是"循环查找"（如果只需点击一次）

### 3. 调试技巧

- ✅ 点击"📂 打开截图"查看截图效果
- ✅ 查看识别结果列表
- ✅ 检查置信度（越高越准确）

---

## 🔍 识别结果显示

### 位置

在按钮下方会显示识别到的文字列表。

### 格式

```
识别到的文字:
• 确定 (置信度: 0.95)
• 取消 (置信度: 0.92)
• 开始 (置信度: 0.88)
```

### 置信度说明

| 置信度 | 说明 |
|--------|------|
| 0.9 - 1.0 | 非常准确 |
| 0.8 - 0.9 | 较准确 |
| 0.7 - 0.8 | 一般 |
| < 0.7 | 可能不准确 |

## ⚠️ 注意事项

### 1. 权限要求

必须授予以下权限：
- ✅ 屏幕录制权限
- ✅ 辅助功能权限

### 2. 性能影响

- 循环查找会持续占用 CPU
- 建议间隔不要小于 1 秒
- 不需要时及时手动停止

### 3. 识别限制

- 只能识别屏幕上可见的文字
- 图片中的文字可能识别不准
- 特殊字体可能识别失败

### 4. 点击限制

- 只能点击当前屏幕上的位置
- 如果窗口移动，需要重新识别
- 多显示器环境可能有问题

## 🆚 两种模式对比

| 特性 | 🔍 查找并点击 | 🔄 循环查找 |
|------|--------------|------------|
| **执行次数** | 1 次 | 持续循环 |
| **自动停止** | ✅ 是 | ❌ 否，需手动停止 |
| **资源占用** | 低 | 中等 |
| **适用场景** | 一次性操作 | 持续监控 |
| **使用间隔** | 不使用 | 使用循环间隔 |
